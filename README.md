# 유니코드 특수 공백 문자 제거기

SMS/MMS에서 공백이 `?`로 표시되는 문제를 해결하는 도구입니다.

## 🚀 빠른 시작 (완성된 실행 파일 다운로드)

GitHub에 코드를 푸시하면 자동으로 빌드됩니다:

1. **GitHub Actions에서 다운로드**:
   - GitHub 저장소의 `Actions` 탭으로 이동
   - 최근 빌드 작업을 선택
   - `Artifacts` 섹션에서 `유니코드제거기-windows` 또는 `유니코드제거기-macos` 다운로드

2. **태그를 사용한 릴리스**:
   - `git tag v1.0.0` (버전 태그 생성)
   - `git push origin v1.0.0` (태그 푸시)
   - GitHub의 `Releases` 탭에서 다운로드 가능

## Windows 실행 파일 빌드 방법

1. Python이 설치되어 있어야 합니다 (Python 3.7 이상)
2. `빌드하기.bat` 또는 `build_windows.bat` 파일을 더블클릭하여 실행
3. 빌드가 완료되면 `dist` 폴더에 `유니코드제거기.exe` 파일이 생성됩니다

## macOS 실행 파일 빌드 방법

1. Python3이 설치되어 있어야 합니다
2. 터미널에서 다음 명령어 실행:
   ```bash
   chmod +x build_macos.sh
   ./build_macos.sh
   ```
3. 빌드가 완료되면 `dist` 폴더에 `유니코드제거기` 실행 파일이 생성됩니다

## 사용 방법

### Windows

#### 방법 1: 실행 파일 직접 실행
- `유니코드제거기.exe`를 더블클릭하여 실행
- 텍스트를 입력하고 빈 줄을 두 번 입력하면 변환된 텍스트가 표시됩니다

#### 방법 2: 파일 처리
- 명령 프롬프트에서: `유니코드제거기.exe 파일명.txt`
- 처리된 텍스트는 `파일명_cleaned.txt`로 저장됩니다

### macOS

#### 방법 1: 실행 파일 직접 실행
- 터미널에서: `./dist/유니코드제거기`
- 텍스트를 입력하고 빈 줄을 두 번 입력하면 변환된 텍스트가 표시됩니다

#### 방법 2: 파일 처리
- 터미널에서: `./dist/유니코드제거기 파일명.txt`
- 처리된 텍스트는 `파일명_cleaned.txt`로 저장됩니다

### Python 스크립트로 실행 (모든 플랫폼)
```bash
python app.py
# 또는
python app.py 파일명.txt
```
